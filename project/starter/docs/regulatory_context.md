# Regulatory Context

The starter project is modeled on common BSA/AML SAR expectations:

- Suspicious activity should be documented clearly enough for investigator follow-up.
- Narratives should explain who acted, what occurred, when it occurred, how much was involved, and why the activity is suspicious.
- Structuring patterns, sanctions exposure, fraud signals, and laundering indicators are treated as distinct typologies.
- Audit trails must preserve reasoning, timestamps, success/failure state, and reviewer decisions.

Common references used in this project:
- `31 CFR 1020.320 (BSA)`
- `12 CFR 21.11 (SAR Filing)`
- `FinCEN SAR Instructions`
